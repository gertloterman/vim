# Vim Configuration

## Features

- File browser
- Syntax checker
- PEP 8 checker
- Bash VI emulator

## Installation

1. Clone the Git repository 

```
git clone git@github.com:gertloterman/vim.git
```

2. Run the install script

```
./install.sh
```

3. Open Vim and run Vundle

```
:PluginInstall
```
